# siriblad

Arbeidsrepo for oppsummering og videre bearbeiding av innhold til Sibirposten jubileum.

Innholdet er basert på dokumentet `Forslag til temaer til Sibirposten Jubileum.docx` og er lagt inn her som et praktisk utgangspunkt for videre redigering.

Start i [docs/sibirposten-jubileum-sammendrag.md](docs/sibirposten-jubileum-sammendrag.md).

Originalt kildemateriale ligger i [source-materiale/forslag-til-temaer-til-sibirposten-jubileum.docx](source-materiale/forslag-til-temaer-til-sibirposten-jubileum.docx).
